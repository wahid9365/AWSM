Asynchronous Transmission mode

    The asynchronous transmission mode provides for communication without time restriction. Packets reach the receiver as fast as possible. For example, protocols of the world wide internet for electronic mail.
    All information of discrete media can be transmitted as an asynchrounous data stream.

    Synchronous Transmission mode

    The synchronous transmission mode defines a maximum end-to-end delay for each packet of a data stream. This upper bound will never be violated. A packet can reach the receiver at any arbitrary earlier time.
    An audio connection can be established over a LAN which supports synchronous transmission mode.

    Isochronous Transmission Mode

    Isochronous transmission mode defines, besides a maximum end-to-end delay fro each packet of a data stream, a minimum end-to-end delay. That means the delay jitter of individual packets is bounded.
