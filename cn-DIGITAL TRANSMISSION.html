DIGITAL TRANSMISSION

  • Data element: Data element is the smallest entity that can represent a piece of information: this is the bit.
  • Signal element: A signal element is the shortest unit (timewise) of a digital signal. In digital data communication, a signal element carries data element.
  • Data Rate: The data rate defines the number of data elements (bits) sent in Is. The unit is bits per second (bps).
  • Signal Rate: The signal rate is the number of signal elements sent in Is. The unit is the baud.
