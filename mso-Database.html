Database

A database is a collection of organized, interrelated data, typically relating to a particular enterprise. Database is representing some aspect of real world which is designed, built and populated with data for a specific purpose.

Characteristics of a Database

  1. It is collection of interrelated data with controlled redundancy
  2. Data is available for all the sub-system of the organization
  3. Accessibility: it may be designed for all sorts of data processing activities viz. batch processing, real time processing etc.
  4. Easy restructuring: the can be modified and updated without much difficulty.
  5. Data independence: data and application programs are independent. So, each of them can be changed without altering the other.
  6. Unauthorized access is restricted
  7. Providing multiple user interfaces
  8. Providing backup and recovery

Types of Database

Operational database:

these database store detailed data needed to support the operations of the entire organization. They are also called as customer database, personnel database, and inventory database etc.

External Database:

Access to a wealth of information from external database is available for a free from commercial online services, and with or without charge from many sources on the internet, especially the World Wide Web. Data are available in the forms of statistics at the data banks

Analytical Database:

these database store data and information extracted from selected operational and external database. They consists of summarized data and information most needed by the end user.

End user database:

these database consists of a variety of data files developed by end user at their workstations. For example, user may have their own copies of documents they download from the www, generated word processing packages or received by electronic mail.

Distributed database:

many organizations distributes copies or parts of the database to network servers at variety of sites. These distributed database can reside on network server on the www, on co-operate intranets or extranets or on other company networks. The distribution of the database is done to improve the database performance and security

Data warehouse:

A data ware house stores the data from current and previous year that has been extracted from the various operation database of an organization. It is a central source of data that has been screened, edited, standardized, integrated so it can be used by managers and other end user professionals for a variety of forms of business analysis, market research and decision support.
