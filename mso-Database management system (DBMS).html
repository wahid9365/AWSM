Database management system (DBMS)

It is a collection of programs that enables user to create and maintain a database. In other words it is general-purpose software that provides the users with the processes of defining, constructing and manipulating the database for various applications. The DBMS contains the information about particular enterprise. It provides an environment that is both convenient and efficient to use. The examples of DBMS are Oracle, IBM DB2 (IBM), SQL Server (Microsoft), Access (Microsoft), MySQL.

The database and DBMS software together is called as Database system. The applications of database systems are in Banking, Airlines, Universities, Sales, and Manufacturing etc.

Objectives of DBMS

  1. Provide for mass storage of relevant data.
  2. Make access to the data easy for user.
  3. Make latest modification to the database available immediately.
  4. Eliminate redundant data.
  5. Allow multiple users to be active at one time.
  6. Provides online maintenance of database.
  7. Allow multiple and concurrent updating and retrievals.

Benefits of DBMS:

  1. Creations of database files.
  2. Definitions of database fields
  3. Easy inclusions of new database fields.
  4. Creation of screens for input and output
  5. Creation of report format.
  6. Rapid sorting and indexing

Difference between file processing system and Database processing system

File processing system Database processing system
Data are separated and isolated Data are integrated
Data duplication results in waste of file space and data integration problem Reduced data duplication
Application programs depend on file format. If file format changes, application programs must also be changed. Data/Programs are independent.
Difficult to represent data in user’s perspective Easier representation of the data in user’s perspective
